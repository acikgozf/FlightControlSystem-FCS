# Ucus_Kontrol_Sistemi



### Features

- RTOS
- Support NMEA and UBX protocol for GNSS (NEO-M8N)
- 6 dof IMU (MPU-6050)
- RX/TX device EBYTE E01-2G4M27D is a DIP module based on originial imported nRF24L01P at 2.4Ghz with 500mW transmitting power.
